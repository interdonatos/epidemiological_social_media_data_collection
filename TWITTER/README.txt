Le dossier 1.ACQUISITION permet d'acquérir automatiquement des tweets mentionnant le terme "avian influenza" dans le texte, grâce à l'API Twitter. 

Le dossier 2.PRE_PROCESSING permet de normaliser le texte des tweets (suppression caractères non unicodes, anonymisation de noms...)

Le dossier 3.ANNOTATION_AUTOMATIQUE_ENTITIES_SPATIALES_ET_THEMATIQUES permet d'annoter automatiquement les entités spatiales (lieux etc.) et les entités thématiques (noms maladies etc.) à la fin des tweets

Le dossier 4.CREATION_XLSX_VISUALISATION_FINALE permet d'obtenir un fichier .xlsx contenant pour une ligne : l'id du tweet, la date de publication, les entités spatiales et thématiques identifiées dans le tweet
